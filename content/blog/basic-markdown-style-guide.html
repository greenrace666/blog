This is a guide to demonstrate HTML syntax for blog posts. You can author content directly in HTML format.

Inline formatting

This text is bold.

This text is italics.

You can strikethrough text.

You can add inline code like this const hello = "world".

Headings

The following HTML heading elements represent five levels of section headings. <h1> is not recommended since the post title is already a <h1> element.

H2: Heading Two

H3: Heading Three

H4: Heading Four

H5: Heading Five
H6: Heading Six

Paragraph

A standalone single paragraph of text.

Paragraphs can be multiline too when they constitute words that make up more than one line, i.e they wrap to the next line. Wow! I am really smart to write two lines of text that makes zero sense.

Blockquotes

This is a blockquote. And it's pretty long too. Long enough to wrap to next line. Surely it will wrap.

You can use other HTML elements like inline code within a blockquote.
